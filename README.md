# Working Excellence — Website Redesign

A fully redesigned single-page website for [workingexcellence.com](https://www.workingexcellence.com).

## What's Built

**Design**
- WEX brand palette: `#f18f0a` orange, full grey scale (`#585b67` → `#b7b9be`)
- Playfair Display + DM Sans + DM Mono type system
- Scroll-triggered fade animations, noise grain overlay, radial glow accents

**Sections**
- Fixed nav with animated mega menu (Practice Areas dropdown with sub-service columns)
- Hero with floating stat cards and image placeholder
- Animated marquee ticker bar
- Problem statement grid (4 pain points)
- Services grid (4 practice areas with image placeholders)
- Assessment products (3 fixed-fee products with deliverables and pricing)
- Interactive cost calculator (cloud waste + AI gap + security exposure)
- Methodology carousel (auto-rotating 4-phase, pause/play, progress bar)
- Competitive comparison table (WEX vs. Big Four vs. MBB)
- Proof stats bar
- Leadership team cards
- Final CTA section
- Footer with full nav
