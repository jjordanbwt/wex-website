# Industry Landing Pages

17 vertical-specific landing pages. All follow the same template.

## Path Rules
These pages live one level deep — use `../` for all shared assets:
- `<link rel="stylesheet" href="../shared.css">`
- `<script src="../shared.js"></script>`
- `<img src="../images/logos/WE_Logo_Orange_White.png">`
- `<link rel="icon" href="../favicon.ico">`

## Adding a New Industry
1. Copy any existing industry page as a template
2. Update `<title>`, meta description, hero headline, and body content
3. Add the new page to the industries mega menu in every page's nav (search `mega-industries`)
4. Add to `industries.html` hub page
